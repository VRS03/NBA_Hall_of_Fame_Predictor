# NBA_Hall_of_Fame_Predictor
Machine learning models trained on historic NBA data to classify current NBA players as future hall of fame candidates or not. 

Models:
- Logistic Regression (baseline)
- Random Forest
- XGBoost

Results:     
                
Logistic Regression: Precision: 0.54 ± 0.03, Recall: 0.84 ± 0.08, F1: 0.66±0.04, ROC-AUC: 0.90±0.04, PR-AUC: 0.47±0.05
                    
Random Forest: Precision: 0.75±0.07, Recall: 0.78±0.07, F1: 0.76±0.06, ROC-AUC: 0.97±0.02, PR-AUC: 0.83±0.07 
                    
XGBoost: Precision: 0.75±0.06, Recall: 0.77±0.08, F1: 0.76±0.06, ROC-AUC: 0.96±0.02, PR-AUC: 0.82±0.07
